# @kitsra/kavio-core

Pure timeline evaluation primitives for Kavio compositions.

## Install

```bash
corepack pnpm add @kitsra/kavio-core
```

## What It Does

- Resolves template props.
- Evaluates layer visibility and local frame timing.
- Interpolates numeric keyframes and easing.
- Resolves layout, anchor, size, and percentage values.
- Evaluates captions, word highlights, and export layer overrides.

`@kitsra/kavio-core` has no browser, FFmpeg, network, or filesystem dependency.
